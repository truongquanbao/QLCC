Thiết kế kiến trúc cho hệ thống WinForms quản lý chung cư với:

- C# .NET 10
- 3-layer architecture:
  Presentation / BLL / DAL

Yêu cầu:

1. Vẽ sơ đồ kiến trúc (text)
2. Mô tả trách nhiệm từng layer
3. Quy tắc giao tiếp giữa các layer
4. Cách tổ chức dependency
5. Logging + exception handling
6. Security (hash password, phân quyền)

Ưu tiên ADO.NET.
