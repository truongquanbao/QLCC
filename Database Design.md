Thiết kế database SQL Server cho hệ thống quản lý chung cư.

Yêu cầu:

1. Danh sách bảng đầy đủ
2. Mô tả từng bảng
3. Khóa chính / khóa ngoại
4. Quan hệ giữa các bảng
5. Chuẩn hóa (3NF)
6. Các constraint quan trọng

Bao gồm:
- Users, Roles, Permissions
- Apartments, Buildings
- Residents
- Invoices, InvoiceDetails
- Complaints
- Vehicles
- Notifications
- Contracts
- AuditLogs
- SystemConfig

Trả lời dạng bảng + mô tả rõ ràng.
